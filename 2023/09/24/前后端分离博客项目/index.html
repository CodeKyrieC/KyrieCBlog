前言

之前学习后端不够系统，很多知识一知半解，虽然做过一些项目，能做出一些东西来，但过于混乱代码不规范且很傻，尤其是sql。

座右铭：迎难而上、永不退缩，抓紧学习的每一分钟。

创建Maven项目（多模块）

准备工作

springboot和mybatispuls整合配置测试

创建启动类 KyrieCblogApplication.java

package cn.kyriec;

import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;

@SpringBootApplication
public class KyrieCblogApplication {
    public static void main(String[] args) {
        SpringApplication.run(KyrieCblogApplication.class);
    }
}

创建springboot配置文件 application.yml

server:
  port: 8888
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/kc_blog?characterEncoding=utf-8&serverTimezone=Asia/Shanghai
    username: root
    password: 123456
    driver-class-name: com.mysql.cj.jdbc.Driver
  servlet:
    multipart:
      max-file-size: 2MB
      max-request-size: 5MB
mybatis-plus:
  configuration:
    log-impl: org.apache.ibatis.logging.stdout.StdOutImpl
  global-config:
    db-config:
      logic-delete-field: delFlag
      logic-delete-value: 1
      logic-not-delete-value: 0
      id-type: auto
